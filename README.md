# Landing Page Project

This project is a simple landing page built using Bootstrap, custom CSS, and JavaScript. It is designed to showcase a collection of listings fetched from a CSV file.

## Project Structure

```
landing-page
├── index.html          # Main HTML file for the landing page
├── css
│   └── styles.css     # Custom styles for the landing page
├── js
│   └── main.js        # JavaScript code for functionality
├── images              # Folder for image files
├── data
│   └── listings.csv    # CSV file containing data for listings
├── vendor
│   └── bootstrap
│       ├── css
│       │   └── bootstrap.min.css  # Minified Bootstrap CSS
│       └── js
│           └── bootstrap.bundle.min.js  # Minified Bootstrap JS
├── .gitignore          # Files and directories to ignore by Git
└── README.md           # Documentation for the project
```

## Setup Instructions

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd landing-page
   ```

3. **Open `index.html` in your web browser** to view the landing page.

## Usage

- The landing page is responsive and built using Bootstrap components.
- Custom styles can be added in `css/styles.css`.
- JavaScript functionality can be modified in `js/main.js`.
- Data for the listings can be updated in `data/icons.csv`.
